Getting Started

This page will quickly walk users through what is required to start using Monero (XMR), including sending/receiving/mining.

  1. Obtain a Monero Wallet. We reccomend the Official Monero Wallet (perhaps using our remote node for convenience), but a nice list of all options can be found here.
  2. Download the mining software and configure it to connect with our pool. This is outlined on the Mining page of our wiki.
  3. View your active miners (rigs) on our website, and configure your settings there as well. This is outlined on the Our Pool page of our wiki.
  4. Learn more about Monero (XMR), its features, and how it works. This is outlined on the Monero (XMR) page of our wiki.

MONERO (XMR) Mining Guide

This page will walk users through the installation and configuration of numerous mining applications.

In order to mine with us, some softare is required. XMR mining is possible with both CPUs and GPUs, however the RandomX algo prevents ASIC miners.

Mining Software

  • XMRig for CPU mining, with GPU (Nvidia & AMD) plugins
  • XMR-Stak for CPU mining

XMRig Configuration

Windows

  1. Download the latest release from Github and extract the files somewhere (if Windows cries about malware or something, exclude the folder in Windows Defender or your Antivirus Software).
  2. With the files extracted, right-click the pool_mine_example file and edit it with notepad.
  3. Replace the line in this file that starts with xmrig.exe with the following, being sure to update your Monero wallet address & a name for your Rig.
    xmrig.exe -o us1.moneromine.co:2222 -u YOUR_WALLET_ADDRESS_HERE --tls -p YOUR_RIG_NAME_HERE
    TLS in command line or json file enables encrypted mining, but you must be on a TLS port for it to work (our port list on the website lists these). Our pool uses the password field in json (or -p in command line) for your Rig's name, an example being: -p MyAwesomeRig.
  4. Save and close notepad.
  5. Right click the pool_mine_example file and Run As Administrator.
  6. Your mining! Your rig should show up on the website once you enter your wallet address (after a minute or two)!

MONEROMINEco Pool Guide

This page will walk users through our pool's features and answer questions around how our pool works.

Our Pool uses PPLNS, which provides fair rewards (shares) for miners and combats pool hopping. We also offer Solo mining ports for those with farm-grade hardware, these miners get %100 of the block reward if/when they find a block. Please note that while we have no pool fee, there is a small TX fee which can be seen under the "Update Threshold" button when configuring your auto-payout amount.

Pool Features

  • 0% Pool Fee
  • Hashrate Capping (25% of global)
  • 0.003 Minimum Payout
  • Block Notify
  • Email Notifications
  • DDoS Protection

Pool Configuration

Auto-Pay Threshold

Changing this number and clicking the "Update Threshhold" button will set when the pool pays you out (Default: 0.3, Minimum: 0.003). Folks who accumulate over 4.0 XMR will not pay the TX fee.

Email Notifications

When adding email for the first time: Simply put your email address in the text field labeled "Change Email To (or initial email)" and click the "Email Alerts" Button.

When changing/updating your email: Simply add your current email address to the text field labeled "Change Email From (current email)", put your new email address in the text field labeled "Change Email To (or initial email)", and click the "Email Alerts" Button.
